# spacegame
Java "Space Game" Demo from https://www.youtube.com/watch?v=hXImR8Wm53M series from RealTutsGML with some enhancements

Includes some code enhancements from the nice video series from RealTutsGML (really cool stuff!). As an example:

- Diffent packages
- Using Generics for controllers
- Enhanced Animation class to accept any number of images using ArrayList<BufferedImage> structure.
- Endless scrolling background
